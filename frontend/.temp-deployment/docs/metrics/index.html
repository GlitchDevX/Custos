Metrics

How to set up a Grafana dashboard for Custos Metrics.

Custos has a Prometheus exporter built in, meaning you can scrape the metrics from Custos using Prometheus and them use them in a Grafana dashboard.

To let Prometheus scrape Custos you can add this simple config to /etc/prometheus/prometheus.yml

prometheus.yml
scrape_configs:
  - job_name: 'custos'
    static_configs:
      - targets: ['localhost:3060']  # assuming your Custos instance is running on port 3060
